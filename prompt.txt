Buatkan project backend modern menggunakan Golang dengan arsitektur modular dan scalable menggunakan framework Fiber pada folder ini.

Backend ini akan digunakan sebagai REST API untuk aplikasi frontend terpisah.

## Tech Stack

* Golang
* Fiber v2
* PostgreSQL
* GORM
* JWT Authentication
* Clean Architecture
* Repository Pattern
* Environment Variables (.env)
* Logger Middleware
* Validation
* Auto Migration

---

# Tujuan Project

Buat backend production-ready dengan struktur modular yang rapi, mudah dikembangkan, dan mengikuti best practice Golang.

# Response Format

Gunakan format response JSON konsisten seperti:

```json
{
  "success": true,
  "message": "Success",
  "data": {}
}
```

Untuk error:

```json
{
  "success": false,
  "message": "Validation error",
  "errors": []
}
```

# Dependensi yang Harus Diinstall

Gunakan command berikut:

```bash
go mod init backend

go get github.com/gofiber/fiber/v2
go get github.com/gofiber/fiber/v2/middleware/logger
go get github.com/gofiber/fiber/v2/middleware/cors
go get github.com/gofiber/fiber/v2/middleware/recover

go get gorm.io/gorm
go get gorm.io/driver/postgres

go get github.com/golang-jwt/jwt/v5
go get golang.org/x/crypto/bcrypt

go get github.com/joho/godotenv
go get github.com/go-playground/validator/v10

go get github.com/google/uuid
go get github.com/rs/zerolog
```

# Konfigurasi Environment

Buat file `.env`

```env
APP_NAME=Fiber Backend
APP_PORT=3000

DB_HOST=localhost
DB_PORT=5432
DB_USER=postgres
DB_PASSWORD=postgres
DB_NAME=fiber_backend

JWT_SECRET=supersecretkey
JWT_EXPIRED=72h
```
